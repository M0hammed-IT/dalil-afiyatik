## 📌 عن المشروع
**دليل عافيتك** هو منصة ويب طبية متكاملة تهدف إلى رقمنة القطاع الصحي وتسهيل وصول المرضى إلى الأطباء والمراكز الصحية. يوفر النظام محرك بحث ذكي، وتفاصيل شفافة عن العيادات (أسعار الكشفيات، أوقات العمل، أرقام الحجز)، بالإضافة إلى قسم ديناميكي مخصص لنشر الوعي والأخبار الطبية.

## ✨ المميزات الرئيسية
- 🔍 **بحث وفلترة ذكية:** إمكانية البحث عن الأطباء حسب التخصص الطبي والمنطقة الجغرافية بدقة وسرعة.
- 👨‍⚕️ **ملفات تعريفية شاملة:** عرض تفاصيل الطبيب، سنوات الخبرة، التقييمات، ومعلومات التواصل المباشر.
- 📰 **قسم الأخبار والنصائح الطبية:** مساحة مخصصة لنشر التوعية الصحية، تحديثات العيادات، وحملات الفحص المجانية.
- 📱 **تصميم متجاوب (Responsive):** تجربة مستخدم (UI/UX) مريحة وسلسة على أجهزة الكمبيوتر والهواتف الذكية.
- ⚙️ **بيانات وهمية جاهزة (Seeding):** مزود بقاعدة بيانات أولية تحتوي على تخصصات وأطباء وأخبار لتجربة النظام مباشرة.

## 🛠️ التقنيات المستخدمة
- **الواجهات الخلفية (Back-End):** PHP, Laravel 
- **الواجهات الأمامية (Front-End):** HTML, CSS, JavaScript, Tailwind CSS
- **قواعد البيانات (Database):** MySQL

## 🚀 طريقة التشغيل محلياً (Local Setup)
لتشغيل المشروع على جهازك الشخصي، يرجى اتباع الخطوات التالية:

1. **نسخ المشروع (Clone):**
   ```bash
   git clone [https://github.com/M0hammed-IT/dalil-afiyatik.git](https://github.com/M0hammed-IT/dalil-afiyatik.git)
2.الدخول إلى مجلد المشروع:
cd dalil_afiyatik

3.تثبيت الحزم البرمجية (Dependencies):
composer install
npm install
npm run build

4.إعداد بيئة العمل وقاعدة البيانات:

قم بنسخ ملف .env.example وأعد تسميته إلى .env.

قم بإنشاء قاعدة بيانات جديدة في (phpMyAdmin) باسم dalil_afiyatik_db.

قم بتحديث بيانات الاتصال في ملف .env لربط قاعدة البيانات.

5.توليد المفتاح وبناء الجداول مع البيانات (Migrate & Seed):
php artisan key:generate
php artisan migrate:fresh --seed

6.تشغيل الخادم المحلي (Serve):
php artisan serve
الموقع سيكون متاحاً الآن عبر الرابط: http://127.0.0.1:8000
